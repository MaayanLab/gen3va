Getting Started

Installing and using GEO2Enrichr

GEO2Enrichr is a web application and browser extension designed to ease the extraction of signatures from studies hosted on the Gene Expression Omnibus (GEO) as well as signatures from your own data sets. You can use the browser extension to extract and label gene signatures from GEO and use the web interface to extract signatures from custom data sets. See GEO2Enrichr's website for more information and detailed instructions.

Tags and reports

A tag is a plain text, non-hierarchical keyword assigned to a gene signature. In social media, the equivalent idea is a "hashtag", which is a tag preceded by a "#" or hash symbol. Tags enable the creation of collections of gene signatures around common themes. A gene signature can have multiple tags. Tagging is handled by GEO2Enrichr and is done when the signature is processed. See GEO2Enrichr's documentation for details.

A report is a GEN3VA page with multiple visualizations and analyses of gene signatures under a single tag. For example, the gene signatures associated with the tag LIGANDS_BD2K_LINCS_DCIC_COURSERA can be found here, while the report is here. A report can also be be generated from a subset of signatures from tags. This can be done using the Custom Report Builder on each tag page.

Understanding the reports

The key idea underpinning GEN3VA is that gene signatures under a common biological theme may share common properties, such as genes in their gene lists, enrichment terms for gene set enrichment analysis, or drug targets. A report consolidates metadata, analyses, and visualizations around a collection of gene signatures around a common theme in an attempt to find consensus between signatures. Reports are best understood as aggregated information. The quality of a report is directly linked to the quality of the metadata and signatures collected, which is an on-going process using GEO2Enrichr.
